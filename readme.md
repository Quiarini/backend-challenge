ENTENDENDO A APLICAÇÃO

A classe TokenValidator realiza a validação de tokens JWT, assegurando que sua estrutura e as claims contenham informações válidas. Através de métodos dedicados, o código verifica a conformidade das claims com regras predefinidas, garantindo a segurança da aplicação que utiliza esses tokens para autenticação e autorização.
